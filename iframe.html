Iframes

Introducción

Un iframe HTML se utiliza para mostrar una página web dentro de una página web.

Sugerencia: es una buena práctica incluir siempre un title atributo para el archivo <iframe>. Los lectores de pantalla lo utilizan para leer cuál es el contenido del iframe.

Sintaxis:

        <iframe src="https://www.google.com/maps?output=embed"></iframe>

Hay páginas como google que no permiten embeber su páginas en otras

Para evitar que mi página se carge en otra

try { console.log('Estamos en iframe: ', window.self !== window.top); console.log('Si nos lo permite CORS, podemos ver la URL:', window.top.location); } catch (e) { /* Las políticas "same origin" impiden que accedamos a window.top */ console.log('Estamos en iframe'); }
