Etiqueta <html>

La etiqueta html indica que es un documento de tipo página web, es la raíz del documento, debe ir en primer lugar (salvo si se indica !Doctype). Entre la etiqueta de apertura y la de cierre deben ir todas las etiquetas de la página web. Es recomendable incluir el atributo lang para indicar el idioma del documento. Este atributo toma como valor el identificativo del lenguaje, por ejemplo, para el español, Lista de códigos de idiomas: ISO-639
