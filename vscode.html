Dar formato automático al Código:

Shift + Alt + F, te tienes que encontrar en la pestaña

También puedes pulsar sobre el menu contextual(botón derecho sobre el código) y elegir la opción dar formato al documento o elegir dar formato al documento con. Uno de los mas famosos es tener instalado el pluging Prettier

Cambiar todas las ocurrencias

Ctrl + F2 Selecciona todas las ocurrencias de la palabra en la que se encuentra el cursor

Ctrl + Shift + L Selecciona todas las ocurrencias del texto seleccionado

Ctrl + D Selecciona unicamente la siguiente ocurrencia del texto seleccionado, si se pulsa repetidamente se amplia la selección

Para Seleccionar la anterior se puede hacer a través del Menu Selección>Agregar repetición anterior o crear un atajo(Agregar selección hasta la anterior coincidencia

Para agregar un atajo: Menu Archivo> Preferencias> Método abreviado del teclado.

Selección de bloques y cursor múltiple

Normalmente cuando seleccionas con el cursor varias líneas, a medida que bajas se seleccionan las líneas completas

Si mientras bajas el cursor pulsas la tecla ALT + Shift verás que ahora se selecciona solamente el área que abarca la posición inicial del curso y la actual, creando una selección rectangular:

Si pulsas Alt + Shift en la misma vertical obtienes un cursor Multiple alineado con esa misma vertical.

Si pulsas Alt mientras vas haciendo click obtienes múltiples cursores.
