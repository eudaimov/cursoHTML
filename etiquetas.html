Elementos

Una página o documento se compone de elementos.

Cada elemento por regla se compone de una etiqueta de comienzo y por regla general, aunque no siempre, de una etiqueta de cierre, un contenido u otros elementos anidados

- La etiqueta de comienzo que va formado por:

  • Un signo llamado antilambda < también llamada diple aviesa de inicio
  • Un nombre de etiqueta, igual en la apertura y en el cierre
  • Unos atributos, de manera opcional en la etiqueta de inicio o apertura
  • Una diple o diplé > de cierre

- Un contenido

- La Etiqueta de cierre compuesta por diple aviesa, una Barra / (escrita de derecha a izquierda y de arriba a abajo), un nombre de etiqueta igual que el de inicio y una diple

Por ejemplo:

<element title="valor del atributo">Contenido</element>

Un elemento puede ser puede anidar otros elementos. Toda página se compone de un árbol de elementos que parte del elemento raíz html

Ejemplo: 

            <body>
                <p>Hola Mundo<p>
            <body>
            <p>Hola
            <b>Mundo<b>
            <p>

Nunca te saltes el elementos de cierre aunque los navegadores sean muy permisibles

Existen elemento vacíos como br y hr se aconseja terminar con />

El estándar no distingue entre mayúsculas y minúsculas aunque recomienda el uso de minúsculas. En antiguos documentos XHTML si que es necesario las minúsculas

Atributos

Los atributos nos proveen de información adicional dentro de las etiquetas

Van colocados dentro de la etiqueta inicial

Tampoco al igual que los nombres de las etiquetas se diferencia entre minúsculas y mayúsculas

Los atributos se definen con una palabra un signo igual y un valor

No es el obligatorio el uso de comillas pero si el valor va formado por dos o mas palabras solo se procesara la primera.

Algunos ejemplos son alt, src, href, title, width, height
