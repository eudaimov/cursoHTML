Listas

Introducción

Con este tipo de etiquetado se pretende clasificar todos aquellos contenidos que pretende ser un listado

Existe tres tripos de listas

  • Listas Ordenadas
  • Listas Desordenadas
  • Lista de términos o conceptos

Creación de Listas Desordenadas

Comienzan con la etiqueta <ul> y terminan por esta misma etiqueta </ul>

Cada elemento de la lista va envuelto a su vez por la etiqueta <li> </li>

Por ejemplo:

            <ul>
                <li>Café</li>
                <li>Leche</li>
                <li>Chocolate</li>
            </ul>

  • Café
  • Leche
  • Chocolate

Para cambiar la forma de la viñeta se emplea dentro del atributo style la propiedad list-style-type:

Los tipos de valores son:

  • disc: Para disco
  • square: Para cuadrado
  • none: Sin viñeta
  • Hay muchos otros: pero se suelen emplear en listas ordenadas, para ver mas pulsa aquí

Creación de listas Ordenadas

Comienzan con la etiqueta <ol> y terminan por esta misma etiqueta </ol>

Cada elemento de la lista va envuelto a su vez por la etiqueta <li> </li>

Por ejemplo:

                <ol>
                    <li>Café</li>
                    <li>Leche</li>
                    <li>Chocolate</li>
                </ol>

  1. Café
  2. Leche
  3. Chocolate

Para cambiar la forma de la numeración se emplea dentro del atributo type, aunque también se puede usar el atributo style la propiedad list-style-type:

Los tipos de valores son:

  • 1: Para números
  • A: Para letras en Mayúsculas
  • a: Para letras en minúsculas
  • I: Sin viñeta
  • i: Para números romanos en minúscula

Para comenzar en un número determinado el atributo start

Lista de conceptos

La<dl>etiqueta define la lista de descripciones, la etiqueta <dt> define el término (nombre) y la etiqueta <dd>describe cada término, es decir, nos da la definición: 

            <dl>
                <dt>Coffee</dt>
                <dd>- black hot drink</dd>
                <dt>Milk</dt>
                <dd>- white cold drink</dd>
              </dl>
Coffee
- black hot drink
Milk
- white cold drink

Contadores CSS

Primer Ejemplo:

                .capitulos{
                    counter-reset: section;
                }
                .capitulo::before{
                    counter-increment: section;
                    content: "Tema " counter(section, upper-roman) "."
                }

Segundo ejemplo:

                .titulo{
                    counter-reset: myseccion;
                }
                .seccion{
                    counter-reset: articulo;
                }
               .seccion::before{
                    counter-increment: myseccion;
                    content: "Sección " counter(myseccion, upper-roman) ".";
                }
                .articulo::before{
                    counter-increment: articulo;
                    content:  counter(myseccion, upper-roman) ". " 
                        counter(articulo) ". Título artículo";
                }

El upper-roman se puede sustituir por todos los valores que se pueden atribuir a la propiedad list-style-type
