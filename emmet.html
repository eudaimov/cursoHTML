Emmet

Es una sintaxis abreviada de código HTML, la mayoría de los editores o lo admiten de serie o se lo puede añadir como si fuera un Plugins

Es muy similar al uso de snippets

Ejemplos

Sintaxis >: Descender en el árbol

Código Emment:

div>ul>li

Resultado:

<div>
    <ul>
        <li></li>
    </ul>
</div>

div+p+blockquote

Notas sobre el formato de abreviaturas

Cuando se familiarice con la sintaxis de las abreviaturas de Emmet, es posible que desee utilizar algún formato para que sus abreviaturas sean más legibles. Por ejemplo, use espacios entre elementos y operadores, así: 

        (header > ul.nav > li*5) + footer

Pero no funcionará, porque el espacio es un símbolo de parada donde Emmet deja de analizar abreviaturas.

Muchos usuarios piensan erróneamente que cada abreviatura debe escribirse en una nueva línea, pero se equivocan: puede escribir y expandir la abreviatura en cualquier parte del texto :
