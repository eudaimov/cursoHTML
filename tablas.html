Tablas

Introducción

Las tablas HTML permiten a los desarrolladores web organizar los datos en filas y columnas.

Una tabla en HTML consta de celdas de tabla dentro de filas y columnas

Para construir una tabla se deben utilizar múltiples elementos:

  • Tabla: <table>
  • Encabezados de Tabla:
  • Cuerpo de la tabla
  • Pie de tabla
  • Filas
  • Celdas de encabezado
  • Celdas
  • Título de Tabla
  • Grupo de columnas
  • Propiedades de columna

Antes de la creación de CSS, los elementos HTML <table> eran frecuentemente utilizados para la disposición (posicionamiento) de una página. Este uso ha sido desalentado desde HTML 4, y el elemento no debe ser usado para dichos fines.

Bordes

Por defecto ninguna tabla tiene un borde definido

¿Como añadir bordes?

        table, th, td {
            border: 1px solid black;
          }

El resultado será el siguiente

Horario Semanal
Lunes Martes Miércoles Jueves Viernes Sábado Domingo
Inglés Matemáticas Literatura Física Geografía Historia Filosofía

¿Cómo colapsar los border?

Para evitar que los border nos aparezcan dobles debemos marcar indicar las siguientes propiedades a través de css

        table, th, td {
            border: 1px solid black;
            border-collapse: collapse;
          }

Otras propiedades de estilos que podemos determinar

border-radius y border-color, border-style, background-color, tr:nth-child(even), tr:nth-child(odd), vertical-align

Tamaños

Las tablas HTML pueden tener diferentes tamaños para cada columna, fila o la tabla completa.

Utilice el atributo style con las propiedades width o height para especificar el tamaño de una tabla, fila o columna.

Nota: Usar un porcentaje como unidad de tamaño para un ancho significa qué tan ancho será este elemento en comparación con su elemento principal, que en este caso es el <body> elemento.

Padding and Spacing

Las tablas HTML pueden ajustar el relleno dentro de las celdas y también el espacio entre las celdas.

Relleno de celda:

El relleno de celda es el espacio entre los bordes de la celda y el contenido de la celda.

De forma predeterminada, el relleno se establece en 0.

>Para agregar relleno en las celdas de la tabla, use la propiedad CSS padding

Espacio entre celdas:

El espacio entre celdas es el espacio entre cada celda.

De forma predeterminada, el espacio se establece en 2 píxeles.

Para cambiar el espacio entre las celdas de la tabla, use la border-spacing propiedad CSS en el table elemento:

Colspan y Rowspan

Las tablas HTML pueden tener celdas que ocupan varias filas y/o columnas.

Para hacer que una celda abarque varias columnas, use el atributo colspan

El valor del atributo colspan representa el número de columnas que abarca.

Para hacer que una celda abarque varias filas, use el atributo:rowspan

El valor del atributo rowspan representa el número de filas que abarca.
