Definición

Según la Real Academia de la lengua HTML es un acrónimo inglés HyperText Markup Language. El cual es: un lenguaje de etiquetas de hipertexto empleado para la elaboración de páginas web, que define una estructura básica y un código (denominado código HTML) para la plasmación del contenido de una página web, texto, imágenes o vídeos, entre otros. Es un estándar a cargo del W3C

Según la w3C:

  • HTML significa lenguaje de marcado de hipertexto
  • HTML es el lenguaje de marcado estándar para crear páginas web.
  • HTML describe la estructura de una página Web
  • HTML consta de una serie de elementos
  • Los elementos HTML le dicen al navegador cómo mostrar el contenido (más bien organiza y clasifica el contenido)
  • Los elementos HTML etiquetan piezas de contenido como "este es un encabezado", "este es un párrafo", "este es un enlace", etc.

Esto quiere decir que la principal función de una página Web es semántica, es decir, dar significado a los contenidos e información que lo componen

Ejemplo:

            <!DOCTYPE html>
            <html>
                <head>
                    <title>Page Title</title>
                </head>
                <body>
                    <h1>This is a Heading</h1>
                    <p>This is a paragraph.</p>
                </body>
            </html>

Explicación

  • El <!DOCTYPE html> Define y determina que el tipo de documento es HTML5
  • The <html> Es el elemento raíz de cualquier página html
  • The <head> Es el elemento que contiene la información relativa a la página, autor, descriptción, título, estilo, script, etc.
  • The <title> element specifies a title for the HTML page (which is shown in the browser's title bar or in the page's tab)
  • The <body> element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
  • The <h1> element defines a large heading
  • The <p> element defines a paragraph
