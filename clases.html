Clases: "class"

Introducción

class es un atributo que se utiliza en html para clasificar elementos en una página html

Múltiples elementos pueden compartir una misma clase.

El valor del atributo es sensible al uso de mayúsculas y minúsculas, no es igual class="negrita" que class="Negrita"

Un elemento html puede tener asignado varias clases separados por espacios

Además de su uso frecuente en css, también es muy frecuente en javascript

Sintaxis:

<style>
 .negrita{
   font-weight:bold;
 }
 .cursiva{
  font-style:italic;
 }
</style>
<p>Lorem ipsum <span class="negrita">dolor</span> sit amet consectetur, adipisicing elit. Quo sapiente expedita, suscipit quibusdam atque est <span class="negrita">porro</span> voluptatibus optio aperiam enim illum consequuntur sint sit ratione iure similique nostrum quisquam ad tempore quidem harum <span class="negrita cursiva">libero</span> maiores voluptate molestiae. Quia culpa expedita, accusamus ab optio voluptatem vel perspiciatis soluta blanditiis, earum dignissimos!</p>
<script>
 document.querySelector(".cursiva").style.color="red";
</script>
