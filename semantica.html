Elementos semánticos, no semánticos y elementos con formato

Elementos semánticos:

Un elemento semántico describe claramente su significado tanto para el navegador como para el desarrollador.

Ejemplos de elementos semánticos : <form>, <table>y <article>- Define claramente su contenido.

Muchos sitios web contienen código HTML como: <div id="nav"> <div class="header"> <div id="footer"> para indicar navegación, encabezado y pie de página. Esto no debería suceder

En HTML hay algunos elementos semánticos que se pueden usar para definir diferentes partes de una página web:

  • <article>
  • <aside>
  • <details>
  • <figcaption>
  • <figure>
  • <footer>
  • <header>
  • <main>
  • <mark>
  • <nav>
  • <section>
  • <summary>
  • <time>

Elementos no semánticos

Ejemplos de elementos no semánticos : <div>y <span>- No dice nada sobre su contenido.

Div es un elemento en bloque cuyo significado no esta definido

Span es un elemento en línea cuyo significado no esta definido.

Elementos con formato

Son elementos cuya función principal es dar una presentación visual a su contenido, algunos de ellos está en desuso o como también se suele decir, son elementos de transición.

Algunos elementos se mezcla la función semántica con la de cierto formato.
