Imágenes

La <img>etiqueta HTML se utiliza para incrustar una imagen en una página web.

Técnicamente, las imágenes no se insertan en una página web; las imágenes están vinculadas a páginas web. La <img>etiqueta crea un espacio de espera para la imagen de referencia.

La <img>etiqueta está vacía, solo contiene atributos y no tiene una etiqueta de cierre. Es aconsejable crear la etiqueta así. <img/>

La <img>etiqueta tiene dos atributos obligatorios:

  • src: especifica la ruta a la imagen
  • alt: especifica un texto alternativo para la imagen

    • Y otros muchos no obligatorios pero de tremenda utilidad.

  • width min and max
  • height min and max
  • object-fit
  • object-position
  • mask-mode
  • mask-mode
  • mask-origin
  • mask-position
  • mask-repeat
  • mask-size
  • opacity
  • image-rendering
  • filter

Ejemplo

<img src="img/earth.png" alt="Planeta Tierra" />

Otras etiquetas relativas a las imágenes:

  • <map> Define un mapa para una imagen.
  • <area> Define areas en las que hacer click dentro de un mapa de imagen.
  • <picture> Define un contenedor para múltiple fuentes de imagen.

    • Picture

            <picture>
            <source media="(max-width:665px)" srcset="img/jupiter.jpg">
            <source media="(max-width:1150px)" srcset="img/earth.png">
            <img src="img/Space1.jpg" alt="Planetas" style="width:auto;">
        </picture>

    Otra forma de colocar imágenes en la web: CSS / background

    Esta manera nos abre nuevas posibilidades como es el uso de sprite y animaciones

        <style>
        #imagen{
            background-image: url("img/earth.png");
            background-size: cover;
            width=500px;
            height=300px;
        }
    </style>
    <div id="imagen"></div>
